# Raspberry Home Server

## Description

Express server to host some websides on the local Network and redirect to other Services on this Pi or on other Devices

### Services

* 127.0.0.1/
* 127.0.0.1/pihole
* 127.0.0.1/octoprint
* 127.0.0.1/door

## Setup

> git clone https://github.com/Dragonil/HomeServer.git

> npm i

> npm start
